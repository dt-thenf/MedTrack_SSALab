# MedTrack — Site Vision & Roadmap

## 1. Vision
MedTrack is a hospital patient monitoring dashboard MVP for doctors and nurses. It provides real-time vital signs monitoring, abnormality detection, alert management, and device connectivity tracking.

**Stitch Project ID:** `1467122588185066807`

## 2. Target Users
- Doctors
- Nurses
- Hospital Staff

## 3. Tech Stack
- HTML/CSS/JS (Tailwind CDN via Stitch generation)
- Inter font (Google Fonts)
- Material Symbols Outlined icons
- Static mock data (MVP phase)

## 4. Sitemap

- [x] `index.html` — Main Dashboard (Hero + Patient Cards + Alerts + Device Status)
- [ ] `patients.html` — Patient List / Detail View
- [ ] `devices.html` — Device Management Page
- [ ] `alerts.html` — Alert History & Management
- [ ] `settings.html` — Settings & Configuration
- [ ] `login.html` — Login / Authentication Screen

1. ~~Dashboard Homepage~~ ✅
